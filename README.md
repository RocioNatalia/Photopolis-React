# PhotoPolis React-App :camera:
Single Page Application para dar like a fotos y almacenarlas en el perfil.
### :heavy_check_mark: Realizada con:
 
 * React JS
 * Create React App
 * React Router
 * Material UI
 * Unsplash API
 * Grid
 * Flaticon
 
### :heavy_check_mark: Para ejecutar localmente: 
```
npm start
```
